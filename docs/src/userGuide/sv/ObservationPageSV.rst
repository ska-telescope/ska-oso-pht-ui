Observation Page
~~~~~~~~~~~~~~~~
On entering the observation page, it will look like :numref:`Figure %s <obspage2>`. 
Science Verifications only allow for a single observation, which is automatically created 
once there is a valid Observing mode ( General Page ) and a valid Target ( Target Page ).

   :width: 90%
   :align: center
   :alt: Image of the Observation page

   Image of the Observation page. 

Layout and Navigation
=====================

This is identical the the observation entry page that is described in the :ref:`Observation Page <observationPage>`.

===========================

The Observation Page allows you to update an observation entry.
